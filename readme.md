# VibeGames API

https://games.h4ks.com

Showroom for shitty vibed projects. Synced to github for easy access and collaboration.

## API
```sh
cd api
poetry install
poetry run python -m app
```

## Frontend
```sh
cd frontend
npm install
npm run dev
```
